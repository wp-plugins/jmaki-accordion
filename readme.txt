== Description ==

Accordion to display links. It allows you to provide multiple panes and display them one at a time. It is like having several CollapsiblePanels where only one can be expanded at a time.

== Installation ==

1. Unzip the accordion.zip
2. Upload accordion to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. In the Plugin Editor click on the jMaki accordion and make changes to reflect your links and titles
